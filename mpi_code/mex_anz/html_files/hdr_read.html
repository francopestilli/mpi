HDR_READ - reads ANALYZE header
    HDR = HDR_READ(FILENAME) reads a header file of ANALYZE(TM).

    For detail, see http://www.mayo.edu/bir/PDF/ANALYZE75.pdf

    REQUIREMENT :
        swapbytes.dll
