HDR_WRITE - create a ANALYZE header file.
    HDR_WRITE(FILENAME,HDR) writes a header file of ANALYZE(TM).
    Input HDR can be given by HDR_INIT.

    For detail, see http://www.mayo.edu/bir/PDF/ANALYZE75.pdf
